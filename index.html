Twine Game: Cupid's Arrows

Download Project Paper (PDF)

Role: Game Level Designer, Lead Writer of Game Story

This is a group project from course CCT270: Principles of Game Design. Our group aimed to develop a game for social change with Twine. Our goal is to create an engaging experience that makes players think critically about the potential dangers of online dating. “Cupid’s Arrows” is a text-based game with interactive elements and multiple endings.

Role: Game Level Designer, Lead Writer of Game Story

This is a group project from course CCT311: Game Design and Theory. Mobile App Prototype: Mate

Download Project Paper (PDF)

Role: User Interface Designer & Developer, Tester

This is a group project from course CCT204: Design Thinking. Our group used Figma to design a music-dating app “Mate” Prototype with multiple functions. Our group followed the principles of design and went through multiple iterations. We created an idea, did the sketches and sent out a survey. Then we developed the prototype, Alpha Version and the final Beta Version based on user feedback and tests, in order to ensure better user experiences. You may send a request to try the prototype in Figma with this link: https://www.figma.com/proto/YuUZotf1yUbTqbdsh3eNgI/Mate?node-id=1-4&starting-point-node-id=1%3A4&t=lx1DSz38I5oii07S-1
